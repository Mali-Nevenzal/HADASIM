## Run Locally

Clone the project

```bash
git clone https://github.com/Mali-Nevenzal/HADASIM/tree/main/CoronaManagementSystem
```

Go to the server directory:

```bash
cd server
```

Install dependencies:

```bash
npm -v
  ```
```bash
npm i nodemon
  ```  
  ```bash
npm i mysql2
  ```
```bash
npm install express
  ```
  ```bash
npm install cors
  ```
Start the server

```bash
npm run start
```
